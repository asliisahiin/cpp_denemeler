# C++ Denemeleri

Bu repo, C++ öğrenirken yazdığım küçük algoritma örneklerini içerir.  
Örneğin üç basamaklı sayıların basamak toplamı, temel döngüler, koşullu ifadeler ve diğer başlangıç seviye alıştırmalar.

## İçerik
- Üç basamaklı sayının basamaklarını toplayan program
- Basit giriş/çıkış örnekleri
- Döngü ve koşul pratikleri
- Diğer temel C++ denemeleri

## Amaç
Bu repo tamamen öğrenme ve pratik amaçlıdır.
